<!--kg-card-begin: markdown--><p><span style="font-size: medium;"><span style="font-family: Calibri;">Furthering Windows Azure commitment to open source development, Microsoft announced <a href="http://www.windowsazure.com/en-us/home/scenarios/mobile-services/">http://www.windowsazure.com/en-us/home/scenarios/mobile-services/</a> that developers can begin using Windows Azure Mobile Services to develop native Android applications.  Mobile Services makes it easy for developers to add a scalable backend powered by Windows Azure to light up their Android phone and tablet apps.  The SDK for Android development, which is now available on </span><a href="https://github.com/WindowsAzure/azure-mobile-services"><span style="color: #0000ff; font-family: Calibri;">GitHub</span></a><span style="font-family: Calibri;">, was developed by Microsoft Open Technologies with the Windows Azure team building the portal integration and push notification support.  </span></span></p>
<p><span style="font-size: medium;"><span style="font-family: Calibri;">These services streamline the development process by enabling developers to use the cloud for backend functions like structured data storage and user authentication via Microsoft account, Facebook, Twitter, and Google.  Whether you are a developer building an app for the Windows Store, iPhone, iPad, or Android and want an easy backend connection for storage, identity and push notifications, Windows Azure Mobile Services provides the right capabilities developers need. </span></span></p>
<p><span style="font-size: medium;"><span style="font-family: Calibri;">Developers will benefit from the Windows Azure Mobile Services update in the following ways: </span></span></p>
<ul>
<li><span style="font-size: medium;"><span style="font-family: Calibri;">Rapid Development: configure a straightforward and secure backend in less than five minutes.</span></span></li>
<li><span style="font-size: medium;"> <span style="font-family: Calibri;">Create modern apps with the following built-in support: </span></span>
<ul>
<li><span style="font-size: medium;"><span style="font-family: Calibri;">Email services through partnership with SendGrid</span></span></li>
<li><span style="font-size: medium;"><span style="font-family: Calibri;">SMS &amp; voice services through partnership with Twilio</span></span></li>
<li><span style="font-size: medium;"><span style="font-family: Calibri;">Access to Structured Storage, Windows Azure Blob, Table, Queues, and ServiceBus  </span></span></li>
<li><span style="font-size: medium;"><span style="font-family: Calibri;">Integrated Authentication so developers can configure user authentication via Microsoft Account, Facebook, Twitter, and Google. </span></span></li>
<li><span style="font-size: medium;"><span style="font-family: Calibri;">Push Notifications for Windows Store, Windows Phone, iOS, and Android apps</span></span></li>
</ul>
</li>
</ul>
<p><span style="font-size: medium;"><strong><span style="font-family: Calibri;">What are you waiting for?</span></strong></span></p>
<p><span style="font-size: medium;"><span style="font-family: Calibri;">Visit <a href="http://windowsazure.com/">WindowsAzure.com</a>, access the current libraries up on <a href="https://github.com/WindowsAzure/azure-mobile-services">GitHub</a>, access the Android Quick Start project in the <a href="http://manage.windowsazure.com">Windows Azure portal</a>, and find <a href="http://www.windowsazure.com/en-us/develop/mobile/tutorials/get-started-android/">tutorials</a> in the Mobile Services dev center. <br /></span><span style="font-family: Calibri;">Windows Azure Mobile Services <em><strong>are still free for your first ten applications running on shared instances</strong></em>. </span></span></p><blockquote class='note original-post'><div><p><strong>Note: </strong>This post originally appeared on my MSDN blog at 		<a href='http://blogs.msdn.com/b/africaapps/archive/2013/03/06/get-started-on-windows-azure-mobile-services-for-windows-store-windows-phone-ios-and-android-apps.aspx'>http://blogs.msdn.com/b/africaapps/archive/2013/03/06/get-started-on-windows-azure-mobile-services-for-windows-store-windows-phone-ios-and-android-apps.aspx</a></p></div></blockquote><!--kg-card-end: markdown-->
