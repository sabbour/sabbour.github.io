<!--kg-card-begin: markdown--><h1>Introduction</h1>  <p>Have you considered migration to a Windows Azure website? The thought of downloading the files and content from your old website, then uploading them again to Windows Azure is not appealing? I agree with you!</p>  <p>The title might be a bit misleading. What we actually want to do is, zip the content of your website (I’m assuming you are using cPanel or something similar and know how to do this), then fetch it directly on the Windows Azure website, and unzip it there.</p>  <p>If you were to deploy that on a Windows Azure Virtual Machine, the above would be very easy, since you either have Remote Desktop access or SSH access to your machine and you can click a few buttons (or punch a few commands) and you will be able to do just that.</p>  <p>The question is, how can you do that without Remote Desktop or SSH access? Since Windows Azure Websites is purely a Platform as a Service, by definition, you have give up access to the underlying machine, in return for higher abstraction. In this post, I’ll show you how to access a “pseudo shell” using the <a href="http://www.visualstudio.com/products/visual-studio-online-overview-vs" target="_blank">cool new Visual Studio Online</a> which is now part of Windows Azure.</p>  <h1>Let’s start</h1>  <p>I assume you have already managed to zip/backup the content of your old website in one way or the other. I will not go through the details of that.</p>  <p>Instead, let’s fire up a new Windows Azure Website from our management portal</p>  <p><a href="__GHOST_URL__/content/images/msdn/6038.image_5F00_12FBA700.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/3678.image_5F00_thumb_5F00_37483D95.png" width="644" height="456" /></a></p>  <p>Once the website is ready, go to the Configure tab, scroll down a bit and enable “<strong>Edit in Visual Studio Online” </strong>then save your settings.</p>  <p><a href="__GHOST_URL__/content/images/msdn/1374.image_5F00_109ED2AA.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/0677.image_5F00_thumb_5F00_43FFB768.png" width="571" height="431" /></a></p>  <p>Once your settings are saved, if you go back to the Dashboard of the website, there should be a new link that comes up on the right under “quick glance” saying ”Edit in Visual Studio Online”. Go ahead an click that link.</p>  <p><a href="__GHOST_URL__/content/images/msdn/8081.image_5F00_5D480680.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/0486.image_5F00_thumb_5F00_54137D52.png" width="313" height="343" /></a></p>  <p>You will be asked to login, use the same username and password you set for the deployment credentials.</p>  <p>Once you are logged in, you will be presented with a beautiful online “IDE” that you can already use to start developing ASP.net, PHP and node.js apps right away!</p>  <p><a href="__GHOST_URL__/content/images/msdn/1565.image_5F00_534038CA.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/2068.image_5F00_thumb_5F00_00102EAA.png" width="644" height="324" /></a></p>  <h1>The console</h1>  <p>The console is the interesting part today, open it by clicking on the second button from the bottom of the left bar, or by pressing <strong>Shift + Ctrl + C</strong> on your keyboard.</p>  <p>If you type <strong>help</strong> in the console, you shall be presented with the list of available commands.</p>  <p><a href="__GHOST_URL__/content/images/msdn/1643.image_5F00_13C239A0.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/4762.image_5F00_thumb_5F00_38E4A56E.png" width="637" height="484" /></a></p>  <p>There are a lot of interesting commands in there, but the one we are going to be using out of those is <strong>unzip</strong>, in order to unzip our backup file once we have it. But how are we going to download the file in the first place? At the time of writing this, there isn’t a command available to download stuff. There are no <strong>wget</strong> or <strong>curl</strong> commands built in.</p>  <p>Fortunately, there is a very easy work around!</p>  <h1>“Installing” cURL</h1>  <p>Turns out you can create .bat files and even run .exe files from this console, so we are in luck. Go ahead and download <a href="http://www.paehl.com/open_source/?CURL_7.33.0" target="_blank">the small cURL zip</a> (I downloaded the <a href="http://www.paehl.com/open_source/?download=curl_733_0_ssl.zip" target="_blank">one with SSL</a> after dismissing that annoying pop-up ad), unzip it on your computer and then simply drag curl.exe from your computer and drop it into the Visual Studio Online app running in your browser <strong>onto the left “Explore” pane</strong>. This will upload it.</p>  <p>Going back to the console and running cURL will work now.</p>  <p><a href="__GHOST_URL__/content/images/msdn/2134.image_5F00_58099158.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/2437.image_5F00_thumb_5F00_2466465C.png" width="644" height="135" /></a></p>  <h1>Migration</h1>  <p>Use cURL to download the zip file from the remote server</p>  <div id="codeSnippetWrapper" style="margin: 20px 0px 10px; padding: 4px; border: 1px solid silver; width: 97.5%; text-align: left; line-height: 12pt; overflow: auto; font-family: &quot;Courier New&quot;, courier, monospace; font-size: 8pt; cursor: text; direction: ltr; max-height: 200px; background-color: rgb(244, 244, 244);">   <div id="codeSnippet" style="padding: 0px; width: 100%; text-align: left; color: black; line-height: 12pt; overflow: visible; font-family: &quot;Courier New&quot;, courier, monospace; font-size: 8pt; direction: ltr; background-color: rgb(244, 244, 244);">     <pre style="margin: 0em; padding: 0px; width: 100%; text-align: left; color: black; line-height: 12pt; overflow: visible; font-family: &quot;Courier New&quot;, courier, monospace; font-size: 8pt; direction: ltr; background-color: white;">curl -o backup.zip http://your.old.website.com/backup.zip</pre>
