<!--kg-card-begin: markdown--><h1>What’s WebMatrix 3?</h1>  <p>If you haven’t checked out <a href="http://www.microsoft.com/web/webmatrix/" target="_blank">WebMatrix 3 yet, you should</a>. It is a free, lightweight, cloud-connected web development tool that enables you to create, publish and maintain your websites with ease. You can use it to create HTML5/CSS3 compliant websites, with LESS and Sass support, andcode completion for JavaScript and jQuery.</p>  <p>You can develop using ASP.net, PHP and Node.js with context sensitive code completion as well.</p>  <p>&amp;#160;</p>  <p><img title="Wordpress specific code completion" alt="Wordpress specific code completion" src="http://www.microsoft.com/web/locale/en-us/media/webmatrix/languages/wm2-php-3.png" width="641" height="342" /></p>  <p>I can go on and on describing the features of WebMatrix 3, but I recommend you just <a href="http://www.microsoft.com/web/webmatrix/" target="_blank">visit the website</a> and check them out yourself.</p>  <h1>Why is it relevant to Windows Azure?</h1>  <p>Well, there are several ways WebMatrix can make your life easier when developing for the cloud. To start with, when you create a new WebSite on Azure, you can quickly open it up in WebMatrix for development and then publish it back through the “Open in WebMatrix” button that shows up in the portal.</p>  <p>&amp;#160;</p>  <p><a href="__GHOST_URL__/content/images/msdn/4745.image_5F00_0EA7ED6D.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/8422.image_5F00_thumb_5F00_47EB27D7.png" width="592" height="484" /></a></p>  <p>&amp;#160;</p>  <p>The second way is when you open up WebMatrix itself, you will be presented with the option to either create a new Windows Azure trial account, or connect your existing one. When you do either, you will be able to directly manage your websites on the cloud from within WebMatrix.</p>  <p><a href="__GHOST_URL__/content/images/msdn/0456.image_5F00_0BDEE622.png"><img title="Opening an existing website" style="display: inline; background-image: none;" border="0" alt="Opening an existing website" src="__GHOST_URL__/content/images/msdn/4807.image_5F00_thumb_5F00_4E21D898.png" width="644" height="470" /></a></p>    <p><a href="__GHOST_URL__/content/images/msdn/8308.image_5F00_0800F5B8.png"><img title="Creating a new website" style="display: inline; background-image: none;" border="0" alt="Creating a new website" src="__GHOST_URL__/content/images/msdn/8203.image_5F00_thumb_5F00_1BEA6281.png" width="644" height="349" /></a></p>  <p>Once you create it, you will be able to work on the local files, then when you save, your changes would be online in seconds!</p>  <p><a href="__GHOST_URL__/content/images/msdn/5415.image_5F00_55C97FA0.png"><img title="image" style="display: inline; background-image: none;" border="0" alt="image" src="__GHOST_URL__/content/images/msdn/8863.image_5F00_thumb_5F00_4DB191A4.png" width="644" height="348" /></a></p>  <p>&amp;#160;</p>  <p>I encourage you to <a href="http://www.microsoft.com/web/webmatrix/" target="_blank">go and try it out</a>.
